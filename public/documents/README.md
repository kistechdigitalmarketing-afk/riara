# Documents Directory

This directory contains downloadable documents such as application forms, fee structures, and prospectuses.

## Required Documents

- `application-form.pdf` - Admission application form
- `fee-structure.pdf` - Current fee structure
- `prospectus.pdf` - School prospectus
- `academic-calendar.pdf` - Academic calendar

## Document Requirements

- All documents should be in PDF format
- File sizes should be optimized (max 5MB recommended)
- Documents should be accessible (text-based PDFs, not scanned images)
- Include proper metadata (title, author, subject)
