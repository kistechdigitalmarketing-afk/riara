# Images Directory

This directory contains all images used on the website. Images should be optimized for web use.

## Directory Structure

```
images/
├── gallery/          # Gallery images (sports, events, activities)
├── news/            # News article images
├── leadership/      # Leadership team photos
├── facilities/      # School facilities photos
├── testimonials/    # Testimonial author photos
└── og-image.jpg     # Open Graph image for social sharing
```

## Image Requirements

- **Format**: JPG or PNG
- **Optimization**: Images should be optimized using tools like ImageOptim or Squoosh
- **Sizes**: 
  - Gallery images: 1200x800px recommended
  - News thumbnails: 800x600px recommended
  - Leadership photos: 600x800px recommended
  - Hero images: 1920x1080px recommended
